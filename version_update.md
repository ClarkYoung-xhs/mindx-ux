# MindX Dashboard 重构与权限中枢升级

## 版本号 / 日期
**v1.5.0** - 2026-03-31

## 核心变更点 (Core Changes)
1. **导航栏底层骨架重组**：打破了原有的单线结构，将导航栏升级为 `Workspace`、`Skills`、`Memory` 三位一体的并列平级结构，大幅提升信息分类层级效率。
2. **重塑 Agent 独立门户**：将原本散落的外置 Agent 模块收罗进全局 `Settings` 窗体中，并采用最高视觉优先级置顶，强化其作为“第二大脑”核心引擎的地位。
3. **扩展 Smart Device 第三方插件专栏**：全面拓宽了右下角的“新建接入（New Source）”弹性版块，在右侧第三列中一次性接入了 10 项数据源快捷入口，涵盖腾讯会议、钉钉、以及所有的录音类智能硬件（统一样式为蓝底带交互滚动条，完美复刻高保真设计图）。
4. **底层架构解耦与付费（PRO）阻断闭环**：对文档详情卡片的更多（MoreVertical）操作菜单进行底层重构，将生硬的中文“导出 PDF”完全废除并改为带有黑底白字 PRO 角标的全局式英文“Convert”，并挂载上了跨组件传导的唤起会员付费升级拦截逻辑。
5. **DOM 层级修复与状态净化**：重写了 Workspace 下处理队列卡片的排版布局，下架了混淆视听的“已提炼”等历史冗余状态，完全聚焦在“提炼中”的核心工作流上，进一步纯净化视觉体验。

## 更新 2026-04-01
1. **聚合文档视图与沉浸式路由闭环**：在 Dashboard 的 Memory 板块中，修复了原先点击独立卡片展示单一信息碎片的孤岛体验。现在，同一大类卡片（例如所有的“关于我”子分类、“当前目标”子目标池）点击后，均会携带聚合参数跳转至 DocumentEditor 以“全览文档”形态合并渲染。同时给后退按钮注入了穿透跳转逻辑，完美实现点哪里进便退回哪里的原生丝滑感。
