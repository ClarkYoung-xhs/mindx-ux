# Workspace 交互功能更新

## 更新日期
2026-04-14

## 功能概述
参考 Notion 的交互设计,为 Workspace 侧边栏添加了更丰富的文档管理功能。

## 主要功能

### 1. ✨ 新建文档入口
- **位置**: Workspace 页面顶部已有"新建文档"按钮
- **支持类型**: 
  - 智能文档 (Smart Canvas)
  - 智能表格 (Smart Sheet)  
  - 页面 (Page)

### 2. 📄 新建子页面功能
- **触发方式**: 鼠标悬停在侧边栏文档项时,显示加号 (+) 按钮
- **规则**:
  - ✅ 文档页面(Smart Canvas/Page)支持新建子页面
  - ❌ 数据表(Smart Sheet)不支持新建子页面
- **默认行为**: 点击加号创建空白的 Smart Canvas 子文档

### 3. 🗂️ 三点菜单操作
- **触发方式**: 鼠标悬停时显示三点菜单 (⋯) 按钮
- **可用操作**:
  1. **创建副本**: 复制当前文档(不包含子文档)
  2. **复制链接**: 复制文档的直接访问链接
  3. **删除**: 删除文档及其子文档

### 4. 📊 数据库快捷入口
- **位置**: 空白 Smart Canvas 文档内
- **功能**: 点击"数据库"按钮可将空白文档转换为 Smart Sheet
- **初始结构**: 自动创建包含"名称"、"状态"、"日期"的基础表格

## 用户体验

### 交互流程示例

1. **创建子页面流程**:
   ```
   悬停文档 → 显示 + 按钮 → 点击 + → 创建空白子页面 → 自动导航到新页面
   ```

2. **转换为数据表流程**:
   ```
   打开空白文档 → 看到"数据库"按钮 → 点击 → 文档转为 Smart Sheet
   ```

3. **删除文档流程**:
   ```
   悬停文档 → 显示 ⋯ 按钮 → 点击 ⋯ → 选择"删除" → 确认 → 删除成功
   ```

## 注意事项

1. ⚠️ **数据表限制**: Smart Sheet 类型的文档不显示加号按钮,无法创建子页面
2. 💾 **数据持久化**: 当前为前端状态管理,刷新后数据会重置(需要后端持久化)
3. 🔄 **副本创建**: 复制功能目前不包含子文档的递归复制
4. 🔗 **链接复制**: 复制的是当前环境的完整 URL
